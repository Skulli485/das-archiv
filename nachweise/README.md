# Nachweise

Hier legst du je Aufgabe eine kurze Datei ab, die echte Ausgaben zeigt, nicht
Beschreibungen von Absichten. Das ist dein Beleg, dass das System trägt.

```
nachweise/
  01-geruest.md      Health durch den Proxy, gewählte Quelle, roher Ausschnitt
  02-adapter.md      curl auf einen Objekt-Aufruf, weggelassene Rohfelder
  03-composable.md   Rückgabe-Form, Renn-Test und Beobachtung
  04-cache.md        netz dann cache mit ms, Health-Modus, Rückfall-Zeile
  05-galerie.md      Miss dann Hit, Tastatur, 390px, eine Grenz-Entscheidung
  06-liefertag.md    Lighthouse-Zahl, git log --oneline, Build-Ausgabe
  07-capstone.md     was gebaut, die Grenz-Entscheidung, ein Test
```

Kurz halten. Eine Terminal-Ausgabe oder ein Screenshot je Punkt reicht.
